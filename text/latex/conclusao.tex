Este trabalho propôs e analisou soluções baseadas em Naïve Bayes para se resolver o problema de classificação de postagens em redes sociais (especificamente no Facebook). Foram estudadas formas de se aliviar as considerações de independência condicional, fazendo uso de pesos para as features.

Os resultados finais obtidos mostraram a viabilidade de se realizar este tipo de classificação, uma vez que foi possível obter acurácias superiores a 60\% (com F1score macro também superior a 60\%) utilizando-se uma base de dados relativamente pequena. Um aumento considerável na base de dados melhoraria muito a performance obtida.

Muitos dos erros cometidos pelo classificador final são justificáveis por terem ocorrido em postagens que na prática poderiam pertencer a mais de uma classe. Deste modo, como trabalho futuro propõe-se a realização de um estudo semelhante com classificadores do tipo \emph{multi-class / multi-label}.
