% Esta dissertação propõe-se a analisar o problema de classificação de textos no contexto de redes sociais (mais especificamente o Facebook). Inicialmente foram implementadas variações de Multinomial Naïve Bayes, utilizando como features as palavras dos textos, posteriormente foram incluídos os metadados das postagens. Como os resultados não foram satisfatórios, explorou-se um modelo de Weighted Naïve Bayes com pesos baseados na Teoria da Informação, o que melhorou significativamente a qualidade dos resultados.

% A partir do classificador criado, desenvolveu-se uma extensão para o navegador Chrome que modifica o feed de notícias do Facebook adicionando um cabeçalho a cada postagem que contem o seu assunto. O usuário pode modificar o assunto, caso discorde da classificação automatizada, dando um feedback que permite uma aprendizagem online para as Redes Bayesianas construídas.

As redes sociais digitais estão cada vez mais presentes na vida das pessoas, ajudando-as a se comunicarem e se manterem informadas. Entretanto, a popularização 
traz também o problema de informações ou postagens indesejadas, o que toma tempo do usuário e torna sua navegação nas redes sociais menos agradável. Dentro 
deste contexto, propõe-se aqui a analisar o problema de classificação de textos no contexto de redes sociais. Foi implementado um classificador baseado em Multinomial Naïve Bayes, utilizando como características as palavras dos textos. O classificador foi integrado à rede social Facebook
através de uma extensão para o navegador Chrome. Esta extensão, desenvolvida no contexto deste trabalho, modifica a linha de notícias, adicionando um cabeçalho à informação de assunto a cada postagem. Permite-se ainda ao usuário modificar o assunto, caso discorde da classificação automatizada. Neste caso, tal informação é utilizada para alimentar a aprendizagem online para as Redes Bayesianas construídas. Em uma segunda versão do classificador, foram incluídos metadados das postagens. O classificador foi treinado e avaliado com bases de dados de postagens e textos classificada por terceiros. Com o intuito de aprimorar o classificador, explorou-se um modelo baseado em Weighted Naïve Bayes, cujos pesos são baseados na Teoria da Informação. Esta técnica trouxe melhoras significativas
na qualidade dos resultados, permitindo acurácia elevada nas diversas situações testadas.
